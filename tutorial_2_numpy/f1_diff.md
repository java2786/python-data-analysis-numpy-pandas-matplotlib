## Numpy is faster than List

Feature | List | Numpy array
--------|------|-------------------
Data type | Mixed | Same type data
Speed | slow | 100 X -> 10 X
Math operations | Need loops | direct work on whole array
Memory usage | large space | low space 
Built-in-math | No | Yes - mean, sum, std, etc.
